## 🛠️ দ্রুত শুরু (30 মিনিট)

### STEP 1: অংশ অর্ডার করুন (1 ঘণ্টা)

**প্রয়োজনীয়:**
- ESP32-CAM (AI-Thinker) - ৳450-550
- DHT11 সেন্সর মডিউল - ৳100-150
- 5mm সবুজ এলীডি × 1 - ৳10
- 5mm লাল এলীডি × 1 - ৳10
- SG90 সার্ভো মোটর - ৳200-250
- 220Ω রেজিস্টর × 5 - ৳10
- জাম্পার ওয়্যার কিট - ৳100-150
- মিনি ব্রেডবোর্ড - ৳50-100
- USB তার (প্রোগ্রামিং) - ৳50-100

**মোট**: ৳1,140-1,730 ✓

**সঙ্গে কিনুন:**
- মাল্টিমিটার (পরীক্ষার জন্য)
- ওয়্যার স্ট্রিপার
- পাওয়ার ব্যাংক (200mA সার্ভো স্পাইকের জন্য)

---

### STEP 2: সফটওয়্যার সেটআপ (15 মিনিট)

**Arduino IDE ডাউনলোড করুন:**
```
1. arduino.cc → Download
2. ইনস্টল করুন
3. খুলুন
```

**এসপি32 বোর্ড যোগ করুন:**
```
Sketch → অন্তর্ভুক্ত করুন → বোর্ড ম্যানেজার
"ESP32" খুঁজুন
"esp32 by Espressif Systems" v2.0.0+ ইনস্টল করুন
```

**লাইব্রেরি যোগ করুন:**
```
Sketch → অন্তর্ভুক্ত করুন → লাইব্রেরি পরিচালনা করুন

খুঁজুন এবং ইনস্টল করুন:
1. "DHT" → Adafruit DHT sensor library
2. "Unified Sensor" → Adafruit Unified Sensor
3. "servo" → ESP32Servo
```

---

### STEP 3: হার্ডওয়্যার সেটআপ (30 মিনিট)

**কানেকশন দ্রুত গাইড:**

```
ব্রেডবোর্ডে:

ESP32-CAM:
├─ GND → নীল রেল (সব GND একসাথে)
├─ 5V → লাল রেল (সব 5V একসাথে)
├─ GPIO 4 → সবুজ এলীডি (220Ω through)
├─ GPIO 12 → লাল এলীডি (220Ω through)
├─ GPIO 13 → সার্ভো হলুদ তার
└─ GPIO 14 → DHT11 ডেটা লাইন

DHT11:
├─ VCC → 5V
├─ DATA → GPIO14
└─ GND → GND

SG90 সার্ভো:
├─ লাল → 5V
├─ বাদামি → GND
└─ হলুদ → GPIO13
```

**বিস্তারিত ডায়াগ্রাম দেখুন:** WIRING_DIAGRAMS.md

---

### STEP 4: কোড আপলোড (10 মিনিট)

```
1. Arduino IDE খুলুন
2. Smart_Room_Automation.ino ফাইল খুলুন
3. Tools → Board → "AI Thinker ESP32-CAM" নির্বাচন করুন
4. Tools → Port → আপনার COM পোর্ট নির্বাচন করুন
5. Sketch → Upload ক্লিক করুন
6. অপেক্ষা করুন (প্রোগ্রাম হয়)
```

**যদি আপলোড ব্যর্থ হয়:**
- USB চালক আপডেট করুন (CH340)
- ভিন্ন USB তার চেষ্টা করুন
- ESP32 এর RST বোতাম চাপুন

---

### STEP 5: পরীক্ষা করুন (5 মিনিট)

```
Arduino IDE খুলুন:
Tools → Serial Monitor (115200 বোড)

প্রত্যাশিত:
✓ "স্মার্ট রুম অটোমেশন শুরু হচ্ছে"
✓ তাপমাত্রা/আর্দ্রতা মূল্য
✓ "সিস্টেম প্রস্তুত"

টেস্ট করুন:
1. 'f' টাইপ করুন → সবুজ এলীডি জ্বলবে
2. ৫ সেকেন্ড অপেক্ষা করুন → স্বয়ংক্রিয় বন্ধ
3. তাপমাত্রা দেখুন → লাল এলীডি পরিবর্তিত হবে
```

## 💡 কীভাবে কাজ করে

### সিস্টেম লজিক

```
প্রতি 2 সেকেন্ড লুপ:

১. DHT11 থেকে তাপমাত্রা পড়ুন
   ↓
२. কক্ষ খালি? → লাল এলীডি বন্ধ রাখুন
   
३. কক্ষ দখলকৃত? → তাপমাত্রা চেক করুন:
   
   • > 32°C → লাল এলীডি ১००% (খুব গরম)
   • २५-३२°C → লাল এলীডি ~३०% (মধ্যম)
   • < २५°C → লাল এলীডি ০% (শীতল)

४. মুখ সনাক্ত? ('f' সিরিয়ালে)
   → সবুজ এলীডি চালু
   → সার্ভো খোলা (९०°)
   → ५ সেকেন্ড টাইমার শুরু

५. ५ সেকেন্ড পরে?
   → সার্ভো বন্ধ (०°)
   → সবুজ এলীডি বন্ধ
```

### GPIO পিন অ্যাসাইনমেন্ট

| পিন | ডিভাইস | ফাংশন |
|-----|--------|--------|
| GPIO 4 | সবুজ এলীডি | ডিজিটাল ON/OFF |
| GPIO 12 | লাল এলীডি | PWM (উজ্জ্বলতা) |
| GPIO 13 | SG90 সার্ভো | PWM (কোণ নিয়ন্ত্রণ) |
| GPIO 14 | DHT11 ডেটা | ডিজিটাল পড়া |
| 5V | পাওয়ার | DHT11, সার্ভো |
| GND | গ্রাউন্ড | সব সংযোগ |
